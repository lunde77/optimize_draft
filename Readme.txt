Draft Optimizer Web Starter

What this includes:
- index.html: clean structure for a free starter webpage
- styles.css: modern dark style
- app.js: local-only interaction (no backend yet)

How to run locally:
1) Open Interface/index.html directly in your browser
   OR
2) From this folder run: python3 -m http.server 8000
   then open http://localhost:8000

Free hosting options:
- GitHub Pages (free for public repos)
- Cloudflare Pages (free tier)
- Netlify (free tier)

Next step to connect optimizer:
